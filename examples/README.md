# Examples

This directory contains framework-specific example catalogs for `@carbon/ai-chat`.

## Choose the example catalog that matches your app

- **React apps**: See [examples/react/README.md](./react/README.md) for React examples built around [`ChatContainer`](../packages/ai-chat/src/react/ChatContainer.tsx:1) and [`ChatCustomElement`](../packages/ai-chat/src/react/ChatCustomElement.tsx:1).
- **Web components apps**: See [examples/web-components/README.md](./web-components/README.md) for examples built around [`cds-aichat-custom-element`](../packages/ai-chat/src/web-components/cds-aichat-custom-element/index.ts:1).

## Which catalog should Angular, Vue, and other framework users use?

If your app uses Angular, Vue, Svelte, or another framework, start with [examples/web-components/README.md](./web-components/README.md). Those examples show the framework-agnostic custom element integration pattern, which is the right starting point outside of React.
