## Overview

Use `cds-aichat-markdown` to render AI responses with a predictable set of Markdown features plus optional HTML sanitization. It supports headings, lists, code blocks with language detection, tables, callouts, and custom link attributes.

### Getting started

Here's a quick example to get you started.

```html
<cds-aichat-markdown>
## Response

Here is a **formatted** answer with a link to the [docs](https://carbondesignsystem.com){target=_blank rel=noopener}.

> Remember to set your API key before calling the service.

* Item one
* Item two
</cds-aichat-markdown>
```
