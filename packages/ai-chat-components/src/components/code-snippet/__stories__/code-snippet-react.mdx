# Code snippet

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/code-snippet) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

The `<CodeSnippet>` component wraps the `cds-aichat-code-snippet` web component to provide syntax highlighting, streaming updates, and editable code blocks inside React applications.

<Canvas of={CodeSnippetReactStories.Default} />

## Usage

Pass the code through the `code` prop (recommended — it bypasses slot change detection, which matters most when streaming):

```jsx
import CodeSnippet from "@carbon/ai-chat-components/es/react/code-snippet.js";

function ReadOnlySnippet({ code }) {
  return <CodeSnippet highlight code={code} />;
}
```

You can also pass code as children, but the `code` prop is more efficient — it bypasses slot change detection, which matters most when streaming.

```jsx
function ReadOnlySnippetChildren({ code }) {
  return <CodeSnippet highlight>{code}</CodeSnippet>;
}
```

For streaming AI responses, append chunks to the `code` prop:

```jsx
function StreamingSnippet({ stream }) {
  const [content, setContent] = React.useState("");

  React.useEffect(() => {
    const subscription = stream.subscribe((chunk) =>
      setContent((previous) => previous + chunk),
    );
    return () => subscription.unsubscribe();
  }, [stream]);

  return <CodeSnippet highlight code={content} />;
}
```

For an editable playground, add `editable` and read `onChange`:

```jsx
function EditableSnippet() {
  const [content, setContent] = React.useState("// Start typing...");

  return (
    <CodeSnippet
      editable
      highlight
      code={content}
      onChange={(event) => setContent(event.detail.content)}
    />
  );
}
```
