# Code snippet

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/code-snippet) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

Use `cds-aichat-code-snippet` to present generated code with syntax highlighting, copy affordances, and optional expand/collapse controls. Wrap it in a card when you need a container.

<Canvas of={CodeSnippetStories.Default} />

## Usage

Pass the code through the `code` attribute (recommended):

```html
<script>
  const code = `import { summarize } from "./summarizer";
export async function summarizeReply(prompt) {
  return (await summarize(prompt)).text;
}`;
</script>

<cds-aichat-code-snippet highlight code="${code}"></cds-aichat-code-snippet>
```

You can also provide code via slotted content, but the `code` attribute is more efficient — it bypasses slot change detection, which matters most when streaming.

```html
<cds-aichat-code-snippet highlight>
  import { summarize } from "./summarizer"; export async function
  summarizeReply(prompt) { return (await summarize(prompt)).text; }
</cds-aichat-code-snippet>
```
