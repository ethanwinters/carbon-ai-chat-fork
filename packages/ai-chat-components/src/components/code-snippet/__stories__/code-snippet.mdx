## Overview

Use `cds-aichat-code-snippet` to present generated code with syntax highlighting, copy affordances, and optional expand/collapse controls. You can wrap it in a `cds-aichat-card` if you need a card container.

### Getting started

Here's a quick example to get you started.

```html
<cds-aichat-code-snippet highlight>
  import { summarize } from "./summarizer";
  export async function summarizeReply(prompt) {
    const result = await summarize(prompt);
    return result.text;
  }
</cds-aichat-code-snippet>
```

Or wrapped in a card:

```html
<cds-aichat-card>
  <cds-aichat-code-snippet highlight>
    import { summarize } from "./summarizer";
    export async function summarizeReply(prompt) {
      const result = await summarize(prompt);
      return result.text;
    }
  </cds-aichat-code-snippet>
</cds-aichat-card>
```
