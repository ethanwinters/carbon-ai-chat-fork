# Card

`<cds-aichat-card>` is an extension of the Carbon Design Systemâ€™s `<cds-tile>` component, designed specifically for AI Chat applications.

## Key Features

- **Rounded corners**: Customizable via `--cds-aichat-card-border-radius` CSS variable
- **Automatic padding removal**: `is-flush` attribute (default: `true`) removes default tile padding
- **OOTB sub-components**: `<cds-aichat-card-footer>` and `<cds-aichat-card-steps>` provide consistent styling

## Customization

> ** For detailed information about the modifier system (`data-rounded`, `data-stacked`) and CSS custom properties, see [Styling & Modifiers](/?path=/docs/introduction-welcome--docs#styling--modifiers).**

### Card Border Radius

Control the card's border radius with the `--cds-aichat-card-border-radius` property:

```css
cds-aichat-card {
  --cds-aichat-card-border-radius: 12px;
}
```

| Property | Default | Description |
| --- | --- | --- |
| `--cds-aichat-card-border-radius` | `0.5rem` (8px) | Card border radius |

**How it works:** The card automatically maps `--cds-aichat-card-border-radius` to `--cds-aichat-rounded-modifier-radius` in its scope, making the value available to slotted content that uses `data-rounded` or reads the modifier properties directly.

### Padding Control

The card removes default tile padding by default via the `is-flush` attribute:

```html
<!-- Default: padding removed -->
<cds-aichat-card></cds-aichat-card>

<!-- Keep tile padding -->
<cds-aichat-card is-flush="false"></cds-aichat-card>
```

**For slotted content**, use `data-flush` to extend content to card edges:

```html
<cds-aichat-card>
  <img slot="media" src="..." data-flush />
  <div slot="body">Content with padding</div>
</cds-aichat-card>
```

| Value | Description |
| --- | --- |
| `data-flush=""` | Removes padding on all sides |
| `data-flush="top"` | Removes top and horizontal padding |
| `data-flush="bottom"` | Removes bottom and horizontal padding |

### Slotted Content Rounding

Use `data-rounded` with slotted content for automatic first/last child rounding:

```html
<cds-aichat-card>
  <div slot="body" data-rounded="bottom" data-stacked>
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3 (rounded bottom corners)</div>
  </div>
</cds-aichat-card>
```

See [Styling & Modifiers](/?path=/docs/introduction-welcome--docs#styling--modifiers) for complete documentation.
