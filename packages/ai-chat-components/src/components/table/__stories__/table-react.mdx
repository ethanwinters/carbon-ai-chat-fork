## Overview

Use `Table` to render structured AI output with filtering, pagination, and loading states in React. It wraps `cds-aichat-table`, so you pass data via props instead of wiring attributes manually.

### Getting started

Here's a quick example to get you started.

```jsx
import Table from "@carbon/ai-chat-components/es/react/table.js";

const headers = [
  { text: "Name" },
  { text: "Role" },
  { text: "Status" },
];

const rows = [
  { cells: [{ text: "Jordan Smith" }, { text: "Designer" }, { text: "Active" }] },
  { cells: [{ text: "Priya Patel" }, { text: "Engineer" }, { text: "Paused" }] },
  { cells: [{ text: "Lee Chen" }, { text: "PM" }, { text: "Active" }] },
];

function TableExample() {
  return (
    <Table
      tableTitle="Agent roster"
      filterPlaceholderText="Filter rows"
      itemsPerPageText="Items per page"
      headers={headers}
      rows={rows}
    />
  );
}
```
