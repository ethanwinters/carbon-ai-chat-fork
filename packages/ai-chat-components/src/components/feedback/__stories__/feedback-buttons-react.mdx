## Overview

Use `<FeedbackButtons>` for quick thumbs up/down capture in React. Pair it with `<Feedback>` to collect richer follow-up when a user picks a sentiment.

## Accessibility

Each button uses `aria-pressed` for its selected state and `aria-controls` to link to its detail panel. Both are keyboard-navigable and expose `positiveLabel`/`negativeLabel` for custom screen-reader text.
