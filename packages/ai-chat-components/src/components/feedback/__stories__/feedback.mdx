## Overview

Use `cds-aichat-feedback` to collect structured feedback on AI responses, with category chips, optional freeform text, and read-only display for feedback already submitted.

## Usage notes

`show-text-area` and `show-body` default to `false`. Set them to enable the text area and body text. `disclaimer` accepts markdown, so you can link out to a privacy policy or terms.
