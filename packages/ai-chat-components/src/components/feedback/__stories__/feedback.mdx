## Overview

Use `cds-aichat-feedback` to collect structured feedback on AI responses. It supports category chips, optional freeform text, prompt copy, and read-only display for previously submitted feedback.

**Note:** The `show-text-area` and `show-body` attributes default to `false`. You must explicitly set them to enable the text area and body text display.

**Note:** When `disclaimer-checkbox` is set, the submit button is disabled until the user checks the checkbox.

### Getting started

Here's a quick example to get you started.

```html
<script type="module">
  const panel = document.querySelector("cds-aichat-feedback");
  panel.categories = ["Accurate", "Helpful", "Too verbose", "Inappropriate"];
  panel.showTextArea = true;  // Enable text area
  panel.addEventListener("feedback-submit", (event) => {
    console.log("Submitted feedback:", event.detail);
  });
  panel.addEventListener("feedback-close", () => {
    console.log("Closed without submitting");
  });
</script>

<cds-aichat-feedback
  id="feedback-panel"
  is-open
  title="Additional feedback"
  body="Why did you choose this rating?"
  text-area-placeholder="Add a comment"
  secondary-label="Cancel"
  primary-label="Submit"
  show-text-area
  show-body
></cds-aichat-feedback>
```

### Legal Disclaimer

Add a legal notice at the bottom of the panel using the `disclaimer` attribute. The text supports markdown.

```html
<cds-aichat-feedback
  disclaimer="By submitting feedback, you agree to our Privacy Policy."
></cds-aichat-feedback>
```

To require the user to explicitly acknowledge the disclaimer before submitting, add a `disclaimer-checkbox` attribute with the checkbox label. The submit button remains **disabled** until the checkbox is checked.

```html
<cds-aichat-feedback
  disclaimer="By submitting feedback, you agree to our Privacy Policy"
  disclaimer-checkbox="I have read and agree to the Privacy Policy."
></cds-aichat-feedback>
```
