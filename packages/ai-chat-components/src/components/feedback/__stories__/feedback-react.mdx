## Overview

Use `<Feedback>` to collect structured feedback on AI responses in React, with categories, optional freeform text, and read-only display for feedback already submitted.

## Usage notes

`showTextArea` and `showBody` default to `false`. Set them to `true` to enable the text area and body text. `disclaimer` accepts markdown, so you can link out to a privacy policy or terms.
