# Video Player

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/video-player) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

The `<VideoPlayer>` component wraps the `cds-aichat-video-player` web component and exposes a React API for playing videos from YouTube, Vimeo, native video files, and Kaltura.

<Canvas of={VideoPlayerReactStories.Default} />

## Key Features

- **Multi-source support**: Automatically detects and handles YouTube, Vimeo, native video files, and Kaltura videos
- **Accessibility**: Built-in keyboard navigation, ARIA labels, and subtitle support
- **Responsive**: Configurable aspect ratio for different video formats
- **Event handling**: React event handlers for play, pause, ready, and error states

## Supported Video Sources

The component automatically detects and handles:

- **YouTube**: `https://www.youtube.com/watch?v=VIDEO_ID` or `https://youtu.be/VIDEO_ID`
- **Vimeo**: `https://vimeo.com/VIDEO_ID`
- **Native files**: Direct links to `.mp4`, `.webm`, `.ogg` files
- **Kaltura**: Kaltura video URLs

## Events

The component supports the following event handlers:

```jsx
import VideoPlayer from "@carbon/ai-chat-components/es/react/video-player.js";

function VideoExample() {
  const handleReady = (event) => {
    console.log("Video is ready to play");
  };

  const handleError = (event) => {
    console.error("Video error:", event.detail);
  };

  return (
    <VideoPlayer
      source="https://www.youtube.com/watch?v=eZ1NizUx9U4"
      onReady={handleReady}
      onError={handleError}
    />
  );
}
```
