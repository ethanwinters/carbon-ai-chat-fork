# Chain of thought

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/chain-of-thought) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

Use `cds-aichat-chain-of-thought` to show the tool calls and intermediate outputs an AI agent used to reach an answer. Pair it with `cds-aichat-chain-of-thought-toggle`, `cds-aichat-chain-of-thought-step`, and `cds-aichat-tool-call-data`; prefer `cds-aichat-reasoning-steps` for non-debugging use cases.
