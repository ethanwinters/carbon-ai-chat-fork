## Overview

Use `cds-aichat-chain-of-thought` to show the tool calls and intermediate outputs an AI agent used to reach an answer. The wrapper simply hosts slotted steps. Pair it with `cds-aichat-chain-of-thought-toggle` for the open/close control, `cds-aichat-chain-of-thought-step` for collapsible steps, and `cds-aichat-tool-call-data` to render tool input/output via slotted markdown content. Prefer `cds-aichat-reasoning-steps` for non-debugging use cases.
