# AI Chat Button (React)

The `<ChatButton>` component wraps the `cds-aichat-button` web component and exposes a React API for use in React applications.

## Key Features

- **Quick Action Mode**: The `is-quick-action` prop renders the button in a compact quick-action style suitable for chat interfaces.
- **Rounded Corners**: Applied by default to match AI chat visual patterns.
- **Preset Defaults**: Quick action buttons automatically enforce recommended `size` and `kind` values.
- **Full Carbon Compatibility**: Supports most of standard `<cds-button>` attributes and events.

## Getting started

Here's a quick example to get you started.

```jsx
import ChatButton from "@carbon/ai-chat-components/es/react/chat-button.js";

function ChatButtonExample() {
  return (
    <ChatButton kind="primary" onClick={() => {}} size="lg">
      Button
    </ChatButton>
  );
}
```
