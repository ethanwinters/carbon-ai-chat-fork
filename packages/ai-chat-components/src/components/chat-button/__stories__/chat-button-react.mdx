## Overview

The `<ChatButton>` component wraps the `cds-aichat-button` web component and exposes a React API for AI chat surfaces, including quick-action and icon-only variants.

### Getting started

Here's a quick example to get you started.

```jsx
import ChatButton from "@carbon/ai-chat-components/es/react/chat-button.js";

function ChatButtonExample() {
  return (
    <ChatButton kind="primary" onClick={() => {}} size="lg">
      Button
    </ChatButton>
  );
}
```
