# AI Chat Button

The `cds-aichat-button` extends the Carbon Design System's [Button](https://web-components.carbondesignsystem.com/?path=/docs/components-button--overview) component with AI Chat-specific features including a quick action mode, and rounded corners designed for chat interfaces.

## Key Features

- **Quick Action Mode**: The `is-quick-action` prop renders the button in a compact quick-action style suitable for chat interfaces.
- **Rounded Corners**: Applied by default to match AI chat visual patterns.
- **Preset Defaults**: Quick action buttons automatically enforce recommended `size` and `kind` values.
- **Full Carbon Compatibility**: Supports most of standard `<cds-button>` attributes and events.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import "@carbon/ai-chat-components/es/components/chat-button/index.js";
```

We provide a lit-react wrapper for this web component to consume in a react application and can be imported as below

```javascript
import ChatButton from "@carbon/ai-chat-components/es/react/chat-button.js";
```
