## Overview

The `<TileContainer>` component wraps the Carbon Design System’s [Tile](https://web-components.carbondesignsystem.com/?path=/docs/components-tile--overview) component to apply **AI Chat–specific styles**.

If you are using a footer inside the container, add the class name `cds-aichat-tile-container-footer` to the footer element to apply the correct footer styling.

---

### Getting started

Here’s a quick example to help you get started.

```jsx
import TileContainer from "@carbon/ai-chat-components/es/react/tile-container.js";
import AiLabel from "@carbon/ai-chat-components/es/react/ai-label.js";
import { Tile, ClickableTile } from "@carbon/ai-chat-components/es/react/tile.js";
import IconButton from "@carbon/ai-chat-components/es/react/icon-button.js";
import Button from "@carbon/ai-chat-components/es/react/button.js";
```

Wrap above  `<Tile>`, `<ClickableTile>` with `<TileContainer>` to get the styles.
The div containing the `cds-aichat-tile-container-footer` will have all the footer styles.

```jsx
<TileContainer>
  <Tile>
    <h5 className="heading-01 margin-bottom-04">
      AI Chat Tile styling wrapper
    </h5>
    <p className="body-01">
      The Carbon Design System provides a comprehensive library of
      components, tokens, and guidelines. We need to implement the new AI
      Chat component following Carbon's design principles and accessibility
      standards.
    </p>
  </Tile>
</TileContainer>
```
