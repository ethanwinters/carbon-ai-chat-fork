# Chat history

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/chat-history) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

`HistoryShell` lays out a chat conversation history panel — header, search toolbar, and a scrollable list of pinned and grouped chats — and provides subcomponents for search results, loading, and delete-confirmation states.

<Canvas of={Stories.Default} />

```jsx
import {
  HistoryShell,
  HistoryHeader,
  HistoryToolbar,
  HistoryContent,
  HistoryPanel,
  HistoryPanelMenu,
  HistoryPanelItem,
  HistoryPanelItems,
} from "@carbon/ai-chat-components/es/react/history";
import { PinFilled, Time } from "@carbon/icons-react";

function ChatHistory() {
  return (
    <HistoryShell>
      <HistoryHeader headerTitle="Chats" />
      <HistoryToolbar />
      <HistoryContent resultsLabel="Results" resultsCount="10">
        <HistoryPanel aria-label="Chat history">
          <HistoryPanelItems>
            <HistoryPanelMenu expanded title="Pinned">
              <PinFilled slot="title-icon" />
              <HistoryPanelItem
                id="chat-1"
                name="Important conversation"
                selected
              />
            </HistoryPanelMenu>
            <HistoryPanelMenu expanded title="Today">
              <Time slot="title-icon" />
              <HistoryPanelItem id="chat-2" name="Recent chat" />
            </HistoryPanelMenu>
          </HistoryPanelItems>
        </HistoryPanel>
      </HistoryContent>
    </HistoryShell>
  );
}
```

### Delete flow

Render `HistoryDeletePanel` when the user chooses **Delete** from a chat item's overflow menu. Pass `itemId` for the chat item being removed. Combined with `HistoryShell`, focus moves to the next chat item after delete, and `history-item-selected` fires on that chat item when the deleted chat item was the active one.

Use `onCancel` to dismiss the panel without deleting, and `onConfirm` to remove the chat from your state and hide the panel.

### Focus management

When pinning or unpinning moves an item between lists, restore focus to it after the DOM updates:

```jsx
import { focusElementAfterRepaint } from "@carbon/ai-chat-components/es/globals/utils/focus-utils";

focusElementAfterRepaint(
  historyShellRef.current ?? document,
  `cds-aichat-history-panel-item[id="${CSS.escape(itemId)}"]`,
);
```
