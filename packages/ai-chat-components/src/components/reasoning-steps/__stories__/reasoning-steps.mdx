# Reasoning steps

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/reasoning-steps) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

Use `cds-aichat-reasoning-steps` to present a concise chain-of-thought timeline of collapsible `cds-aichat-reasoning-step` entries, auto-managed or fully controlled via events.

<Canvas of={ReasoningStepsStories.Default} />

### External toggle

Pair it with `cds-aichat-reasoning-steps-toggle` for a ready-made external toggle control.

### With static steps

Steps without body content render as a static line, useful for work that is still pending or awaiting input.

<Canvas of={ReasoningStepsStories.WithStaticSteps} />

### Controlled

When `controlled` is set, the host application owns each step's open state and responds to `reasoning-step-beingtoggled` events.

<Canvas of={ReasoningStepsStories.Controlled} />
