## Overview

Use `cds-aichat-reasoning-steps` to present a concise chain-of-thought timeline. Each child `cds-aichat-reasoning-step` can collapse or expand, and you can let the component auto-manage state or fully control it via events when determinism is required.

Pair it with `cds-aichat-reasoning-steps-toggle` if you want a ready-made control for opening and closing the wrapper.

### Getting started

Here's a quick example to get you started.

```html
<script type="module">
  const steps = document.querySelector("cds-aichat-reasoning-steps");
  steps.addEventListener("cds-aichat-reasoning-step-toggled", (event) => {
    const title = event.target?.getAttribute("title") || "Reasoning step";
    console.log(`${title} is now ${event.detail.open ? "open" : "closed"}`);
  });
</script>

<cds-aichat-reasoning-steps open>
  <cds-aichat-reasoning-step title="Understand the request" open>
    Parsed the prompt, captured constraints, and defined the objective.
  </cds-aichat-reasoning-step>
  <cds-aichat-reasoning-step title="Review retrieved context">
    Checked docs and history for facts that support the objective.
  </cds-aichat-reasoning-step>
  <cds-aichat-reasoning-step title="Draft an answer">
    Combined the prompt with trusted context and drafted a response.
  </cds-aichat-reasoning-step>
</cds-aichat-reasoning-steps>
```
