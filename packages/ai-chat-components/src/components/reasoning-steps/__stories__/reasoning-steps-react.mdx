# Reasoning steps

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/reasoning-steps) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

The `<ReasoningSteps>` and `<ReasoningStep>` components wrap the underlying web components so you can show chain-of-thought style reasoning in React applications without managing event wiring manually.

<Canvas of={ReasoningStepsReactStories.Default} />

### With static steps

Steps without body content render as a static line, useful for work that is still pending or awaiting input.

<Canvas of={ReasoningStepsReactStories.WithStaticSteps} />

### Controlled

When `controlled` is set, the host application owns each step's open state and responds to `onToggle` events.

<Canvas of={ReasoningStepsReactStories.Controlled} />
