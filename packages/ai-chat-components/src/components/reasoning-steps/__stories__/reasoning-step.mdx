## Overview

`cds-aichat-reasoning-step` is the building block for the reasoning timeline. It renders a title with optional body content, supports controlled open state, and gracefully collapses to a static line when no body is provided.

### Getting started

Here's a quick example to get you started.

```html
<cds-aichat-reasoning-steps controlled open>
  <cds-aichat-reasoning-step title="Gather context" controlled open>
    Pulled CRM notes, entitlement data, and recent tickets tied to this user.
  </cds-aichat-reasoning-step controlled>
  <cds-aichat-reasoning-step title="Flagged missing data" controlled>
  </cds-aichat-reasoning-step>
</cds-aichat-reasoning-steps>
```
