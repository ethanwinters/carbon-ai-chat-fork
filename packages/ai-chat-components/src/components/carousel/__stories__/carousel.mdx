# Carousel

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/carousel) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

Use `cds-aichat-carousel` to scroll a set of slotted items, such as cards, through a paged viewport with next/previous controls. It's built on the `@carbon/utilities` carousel primitive — see the [carbon core carousel docs](https://github.com/carbon-design-system/carbon/tree/main/packages/utilities/src/carousel) for implementation details.
