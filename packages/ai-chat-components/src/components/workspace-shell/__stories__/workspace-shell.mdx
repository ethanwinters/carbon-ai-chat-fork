## Overview

The **`<cds-aichat-workspace-shell>`** component defines five dedicated slots that structure the overall workspace layout:

<table style={{width: '100%'}}>
  <thead>
    <tr>
      <th>Slot Name</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>toolbar</code></td>
      <td>Represents the toolbar area of the workspace.</td>
    </tr>
    <tr>
      <td><code>header</code></td>
      <td>Represents the header section, containing title, subtitle and actions.</td>
    </tr>
    <tr>
      <td><code>notification</code></td>
      <td>Area for displaying workspace notifications.</td>
    </tr>
    <tr>
      <td><code>body</code></td>
      <td>The main content area of the workspace.</td>
    </tr>
    <tr>
      <td><code>footer</code></td>
      <td>Represents the footer section, usually containing action buttons.</td>
    </tr>
  </tbody>
</table>


Each of these slots can accept **any element or component**, allowing users to freely construct and customize the workspace layout as needed.
Alternatively, adopters can use the **provided subcomponents**, which are designed specifically for each section. These subcomponents offer consistent styling and default behaviors that align with the AI Chat Workspace design patterns.

Using these subcomponents ensures visual consistency and predictable structure while still giving you the flexibility to build a fully customized workspace experience.

### Layout Tokens

When used inside `cds-aichat-shell`, the workspace and history panel widths are controlled by chat shell layout tokens. See the Chat shell story for the full layout token list.

If you are using this component with `@carbon/ai-chat` (most common) you can apply these CSS variables anywhere on your page.

```css
cds-aichat-shell {
  --cds-aichat-workspace-min-width: 520px;
  --cds-aichat-history-width: 360px;
}
```

| Token | Default | Description |
| --- | --- | --- |
| `--cds-aichat-workspace-min-width` | `480px` | Minimum width for workspace panel |
| `--cds-aichat-history-width` | `320px` | Width of the history / conversation list panel |
