# Workspace shell

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/workspace-shell) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

Use `WorkspaceShell` to lay out a workspace panel's toolbar, notification, header, body, and footer into a single consistent structure.

<Canvas of={WorkspaceShellStories.Default} />

`WorkspaceShell` defines five dedicated slots that structure the overall workspace layout: `toolbar`, `header`, `notification`, `body`, and `footer`. Each slot accepts any element or component, or you can use the provided subcomponents (`WorkspaceShellHeader`, `WorkspaceShellBody`, `WorkspaceShellFooter`) for consistent styling and default behaviors that align with the AI Chat Workspace design patterns.

## JS (via import)

```javascript
import WorkspaceShell, {
  WorkspaceShellHeader,
  WorkspaceShellBody,
  WorkspaceShellFooter,
} from "@carbon/ai-chat-components/es/react/workspace-shell.js";
```

## Customization

> ** For information about styling slotted content with modifiers (`data-rounded`, `data-stacked`) and CSS custom properties, see [Styling & Modifiers](/docs/introduction-styling-modifiers--overview).**

### Layout Tokens

When used inside `ChatShell`, the workspace and history panel widths are controlled by chat shell layout tokens. See the Chat shell story for the full layout token list.

If you are using this component with `@carbon/ai-chat` (most common) you can apply these CSS variables anywhere on your page.

```jsx
<div
  style={{
    "--cds-aichat-workspace-min-width": "520px",
  }}
>
  <ChatShell showFrame showWorkspace>
    <WorkspaceShell>
      <WorkspaceShellBody>Workspace content</WorkspaceShellBody>
    </WorkspaceShell>
  </ChatShell>
</div>
```

| Token                              | Default | Description                                                                                                                     |
| ---------------------------------- | ------- | ------------------------------------------------------------------------------------------------------------------------------- |
| `--cds-aichat-workspace-min-width` | `480px` | Minimum width for workspace panel. Will switch from inline to an overlay panel if it can't render to at least this size inline. |
