# Transcript

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/audio-player) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

Use `cds-aichat-transcript` to give audio and video content an accessible, expandable/collapsible text alternative.

<Canvas of={Stories.Default} />

## Key Features

- **Expandable/collapsible**: Clean interface that expands on demand
- **Markdown support**: Render formatted transcript text
- **Accessibility**: Proper ARIA attributes and keyboard navigation
- **Event handling**: Custom events for expand/collapse state changes
- **Internationalization**: Support for multiple languages

## Events

`expanded` is a controlled property — the component only dispatches the toggle event below, so the host is responsible for setting `expanded` back on the element:

```javascript
const transcript = document.querySelector("cds-aichat-transcript");

transcript.addEventListener("cds-aichat-transcript-toggle", (event) => {
  transcript.expanded = event.detail.expanded;
});
```
