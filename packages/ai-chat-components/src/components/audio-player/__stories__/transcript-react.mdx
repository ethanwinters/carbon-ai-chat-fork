# Transcript

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/audio-player) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

The `<Transcript>` component wraps `cds-aichat-transcript` to give audio and video content an accessible, expandable/collapsible text alternative.

<Canvas of={Stories.Default} />

## Key Features

- **Expandable/collapsible**: Clean interface that expands on demand
- **Markdown support**: Render formatted transcript text
- **Accessibility**: Proper ARIA attributes and keyboard navigation
- **Event handling**: React-friendly event callbacks
- **Internationalization**: Support for multiple languages

## Events

`expanded` is a controlled prop — the component only fires the event below, so the host is responsible for setting `expanded` back on the element:

```jsx
import { Transcript } from "@carbon/ai-chat-components/es/react/transcript";
import { useState } from "react";

function TranscriptWithState() {
  const [isExpanded, setIsExpanded] = useState(false);

  return (
    <Transcript
      text="Transcript text here..."
      label="English Transcript"
      language="en"
      expanded={isExpanded}
      onTranscriptToggle={(e) => {
        setIsExpanded(e.detail.expanded);
      }}
    />
  );
}
```

| Event Prop           | Event Name                     | Description                                    | Event Detail            |
| -------------------- | ------------------------------ | ---------------------------------------------- | ----------------------- |
| `onTranscriptToggle` | `cds-aichat-transcript-toggle` | Fired when transcript is expanded or collapsed | `{ expanded: boolean }` |
