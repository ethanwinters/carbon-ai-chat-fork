# Audio Player

`<cds-aichat-audio-player>` provides a unified interface for playing audio from multiple sources including SoundCloud, native audio files, and more.

## Key Features

- **Multi-source support**: Automatically detects and handles SoundCloud and native audio files
- **Accessibility**: Built-in keyboard navigation, ARIA labels, and transcript support
- **Responsive**: Configurable aspect ratio for visual consistency
- **Event handling**: Custom events for play, pause, ready, and error states

## Supported Audio Sources

The component automatically detects and handles:

- **SoundCloud**: `https://soundcloud.com/...`
- **Native files**: Direct links to `.mp3`, `.wav`, `.ogg`, `.m4a` files

## Getting started

Here's a quick example to get you started.

```html
<cds-aichat-audio-player
  source="https://soundcloud.com/ibmthinkleaders/leveraging-ai-to-tackle-large-problems-being-an-optimistic-futurist-feat-kate-oneill"
  title="Audio Title"
  description="Audio description"
  aria-label="Podcast episode"
>
</cds-aichat-audio-player>
```

## Events

The component fires the following custom events:

```javascript
const player = document.querySelector('cds-aichat-audio-player');

player.addEventListener('cds-aichat-audio-player-ready', (event) => {
  console.log('Audio is ready to play');
});

player.addEventListener('cds-aichat-audio-player-error', (event) => {
  console.error('Audio error:', event.detail);
});
```
