# Audio Player

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/audio-player) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

The `<AudioPlayer>` component wraps `cds-aichat-audio-player` to play audio from SoundCloud or native audio files behind a single, accessible React API.

<Canvas of={Stories.Default} />

## Key Features

- **Multi-source support**: Automatically detects and handles SoundCloud and native audio files
- **Accessibility**: Built-in keyboard navigation and ARIA labels
- **Event handling**: React event handlers for play, pause, ready, and error states

## Supported Audio Sources

The component automatically detects and handles:

- **SoundCloud**: `https://soundcloud.com/...`
- **Native files**: Direct links to `.mp3`, `.wav`, `.oga`, `.m4a` files

## Events

The component supports the following event handlers:

```jsx
import AudioPlayer from "@carbon/ai-chat-components/es/react/audio-player.js";

function AudioExample() {
  const handleReady = (event) => {
    console.log("Audio is ready to play");
  };

  const handleError = (event) => {
    console.error("Audio error:", event.detail);
  };

  return (
    <AudioPlayer
      source="https://soundcloud.com/ibmthinkleaders/leveraging-ai-to-tackle-large-problems-being-an-optimistic-futurist-feat-kate-oneill"
      onReady={handleReady}
      onError={handleError}
    />
  );
}
```
