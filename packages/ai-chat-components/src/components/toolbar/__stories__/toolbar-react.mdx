# Toolbar

[Source code](https://github.com/carbon-design-system/carbon-ai-chat/tree/main/packages/ai-chat-components/src/components/toolbar) &nbsp;|&nbsp; [Feedback](https://github.com/carbon-design-system/carbon-ai-chat/issues)

## Overview

`<Toolbar>` aligns titles, labels, icon buttons, and overflow actions into a responsive header for AI chat surfaces. It handles truncation and optional overflow menus so you can focus on wiring your actions.

<Canvas of={ToolbarStories.Default} />

## Customization

> **For information about styling slotted content with modifiers like `data-rounded`, see [Styling & Modifiers](/docs/introduction-styling-modifiers--overview).**

Slotted content can use `data-rounded` to align with corner rounding:

```jsx
<Toolbar>
  <div slot="navigation" data-rounded="top-left">
    {/* Content gets top-left corner rounding */}
  </div>
</Toolbar>
```
