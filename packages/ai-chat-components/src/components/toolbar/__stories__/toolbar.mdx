## Overview

`cds-aichat-toolbar` provides a flexible header bar for AI chat surfaces, aligning titles, labels, icon buttons, and overflow actions while handling truncation and responsive wrapping for you.

### Getting started

Here's a quick example to get you started.

```html
<cds-aichat-toolbar overflow .actions="${actions}">
  <!-- Navigation slot -->
  <div slot="navigation" data-fixed="" data-rounded="top-left">
    <cds-icon-button
      kind="ghost"
      @click=${action("onClick")}
      align="bottom-start"
      enter-delay-ms="0"
      leave-delay-ms="0"
    >
      ${iconLoader(Home16, { slot: "icon" })}
      <span slot="tooltip-content">Home</span>
    </cds-icon-button>
  </div>
```
