# AI Chat Toolbar

The `cds-aichat-toolbar` builds on the Carbon Design System's icon buttons, coupled with overflowHandler from @carbon/utilities to create a responsive toolbar for AI Chat applications.

## Key Features

- **Overflow**: A property/attribute to opt in to collapse actions that don't fit in the toolbar into an overflow menu.
- **Slots**: Supports slots for navigation, title, fixed actions, and AI label.
- **Rounded corners**: Makes use of rounded modifier scss utility. and can be controlled via `--cds-aichat-rounded-modifier-radius` CSS variable.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import "@carbon/ai-chat-components/es/components/toolbar/index.js";
```

We provide a lit-react wrapper for this web component to consume in a react application and can be imported as below

```javascript
import Toolbar from "@carbon/ai-chat-components/es/react/toolbar.js";
```

### HTML

```html
<cds-aichat-toolbar overflow="" .actions="${actions}">
  <!-- Navigation slot -->
  <div slot="navigation" data-fixed="" data-rounded="top-left">
    <cds-icon-button
      kind="ghost"
      @click=${action("onClick")}
      align="bottom-start"
      enter-delay-ms="0"
      leave-delay-ms="0"
    >
      ${iconLoader(Home16, { slot: "icon" })}
      <span slot="tooltip-content">Home</span>
    </cds-icon-button>
  </div>

  <!-- Title slot -->
  <div slot="title" data-fixed="">Title <span class="bold">text</span></div>

  <!-- Fixed actions slot -->
  <div slot="fixed-actions" data-fixed="">
    <cds-button size="md">test</cds-button>
  </div>

  <!-- AI Label slot -->
  <cds-ai-label
    size="2xs"
    autoalign=""
    alignment="bottom"
    slot="toolbar-ai-label"
  >
    <div slot="body-text">
      <h4 class="margin-bottom-05">Powered by IBM watsonx</h4>
      <div>
        IBM watsonx is powered by the latest AI models to intelligently process
        conversations and provide help whenever and wherever you may need it.
      </div>
    </div>
  </cds-ai-label>
</cds-aichat-toolbar>
```
