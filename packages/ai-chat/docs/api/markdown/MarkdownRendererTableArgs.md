# MarkdownRendererTableArgs

**Experimental.**

- Kind: TypeAlias
- Category: Messaging
- Reference: https://chat.carbondesignsystem.com/tag/latest/docs/types/Type_reference.MarkdownRendererTableArgs.html

Argument passed to the markdown table renderer callbacks on
CustomMarkdownRenderers.table and
WCCustomMarkdownRenderers.table. Extends
MarkdownRendererTableData with the source token, full
TokenTree node, and a stable `slotName` suitable for use as a key.

type MarkdownRendererTableArgs = _MarkdownRendererTableArgs
