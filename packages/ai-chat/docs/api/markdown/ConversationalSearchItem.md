# ConversationalSearchItem

- Kind: Interface
- Category: Messaging
- Reference: https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.ConversationalSearchItem.html

A text response generated by AI with an optional list of citations for where the information came from.

## Signature

```ts
interface ConversationalSearchItem
```

## Members

### agent_message_type

`agent_message_type?: HumanAgentMessageType`

For messages that are sent between the user and a human agent, we assign an agent type to the message to distinguish what type it is.

[Reference](https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.ConversationalSearchItem.html#agent_message_type)

### citations

`citations?: ConversationalSearchItemCitation[]`

Citations are used to connect specific text within a conversational search response with the relevant documents
returned by the backend.

[Reference](https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.ConversationalSearchItem.html#citations)

### response_type

`response_type: MessageResponseTypes`

The response type of this message item.

[Reference](https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.ConversationalSearchItem.html#response_type)

### search_results

`search_results?: SearchResult[]`

In some implementations, the content searched isn't in an accessible URL. For instance, it could be from Milvus or
ElasticSearch. In that scenario, you may populate "search_results". Combine this with
ConversationalSearchItemCitation.search_result_idx.

[Reference](https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.ConversationalSearchItem.html#search_results)

### streaming_metadata

`streaming_metadata?: ItemStreamingMetadata`

Metadata used to identify a generic item within the context of a stream in order to correlate any updates meant
for a specific item.

[Reference](https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.ConversationalSearchItem.html#streaming_metadata)

`text: string`
