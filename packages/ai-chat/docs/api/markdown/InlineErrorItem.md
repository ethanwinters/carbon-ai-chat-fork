# InlineErrorItem

- Kind: Interface
- Category: Messaging
- Reference: https://chat.carbondesignsystem.com/version/v1.18.0-rc.0/docs/interfaces/Type_reference.InlineErrorItem.html

An inline error response generated by a conversational skill provider with a user-friendly text and developer debug info.

## Signature

```ts
interface InlineErrorItem
```

[Reference](https://chat.carbondesignsystem.com/version/v1.18.0-rc.0/docs/interfaces/Type_reference.InlineErrorItem.html#agent_message_type)

### debug

`debug?: { info?: Record<string, unknown>; statusCode?: number; text?: string }`
