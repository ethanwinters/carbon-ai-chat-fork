# LayoutCustomProperties

- Kind: Enum
- Category: Config
- Reference: https://chat.carbondesignsystem.com/tag/latest/docs/enums/Type_reference.LayoutCustomProperties.html

These variables map to CSS custom properties used in styling the AI chat interface.

Keys map to the underlying `--cds-aichat-*` custom properties.

You can use any standard CSS as the value.

## Signature

```ts
enum LayoutCustomProperties
```
