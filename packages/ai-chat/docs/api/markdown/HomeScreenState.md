# HomeScreenState

- Kind: Interface
- Category: Config
- Reference: https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.HomeScreenState.html

Current state of home screen (currently, limited to if it is open or closed).

## Signature

```ts
interface HomeScreenState
```

## Members

### isHomeScreenOpen

`isHomeScreenOpen: boolean`

Indicates if the home screen is currently open.

[Reference](https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.HomeScreenState.html#ishomescreenopen)

### showBackToAssistant

`showBackToAssistant: boolean`

Indicates if the home screen should display a "return to assistant" button. This button is displayed when the user
has clicked the "back to home" button from the assistant.

[Reference](https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.HomeScreenState.html#showbacktoassistant)
