# MediaTranscript

- Kind: Interface
- Category: Messaging
- Reference: https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.MediaTranscript.html

Represents a text transcript for audio content.
Displayed as readable text below the audio player for accessibility.

## Signature

```ts
interface MediaTranscript
```

## Members

### label

`label?: string`

Optional label for the transcript (e.g., "English Transcript").

[Reference](https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.MediaTranscript.html#label)

### language

`language?: string`

Language of the transcript (e.g., "en", "es", "fr").

[Reference](https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.MediaTranscript.html#language)

### text

`text: string`

Full text transcript of the audio content.
Supports markdown for formatting.

[Reference](https://chat.carbondesignsystem.com/tag/latest/docs/interfaces/Type_reference.MediaTranscript.html#text)
